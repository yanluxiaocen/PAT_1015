# PAT_1015
基础题

今天咱打的不是败战！

虽然回首看去，这题就是个纯分级排序题，客观看除了代码多些没啥难点。

但其实这里面对于数组和结构的用法，没有这段时间刷的题积累经验，我根本写不出来，额...虽然好像也没怎么涉及经验，那就换成思路。

过去看到这种题目哪怕看得明白，有思路，但往往难以实现，有坚持到一半后就力竭和难以专注继续的事情发生...所以换句话说，其实是给我增添了更多信心吧？

回归正题，之所以记录此题，不只是为了庆祝胜利，还为了增加头文件为<stdlib.h>的快速排序函数qsort。

它的语法为qsort(数组首地址, 元素个数, 每个元素的大小, 比较函数);

qsort能有效替代冒泡算法，我在这题原本用的就是冒泡算法，但奈何元素太多，100000个，提交结果有两个死活运行超时。

说到冒泡算法，我以前其实不怎么喜欢它，因为代码冗杂，而且笨重，但它的思路清晰是我现在最喜欢的一点，所以最近凡是需要排序的，我都用冒泡一口气给它填上了。

总之我对今天的结果还算满意，再接再厉吧
